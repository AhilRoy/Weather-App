# Weather-App
A Python GUI weather application using Open Weather API

## 📌 Features

- 🌍 Search weather by city name
- 🌡 Displays current temperature
- 🥵 Shows feels-like temperature
- 🔺 Maximum and minimum temperature
- 💧 Humidity information
- 📈 Atmospheric pressure details
- ☁️ Weather condition and description
- 💨 Wind speed and direction
- 👀 Visibility information
- 📍 Latitude and longitude details
- 🌅 Sunrise and sunset timings
- 📜 Scrollable weather report interface
- 🖥 Standalone Windows application (.exe)

## 🛠 Technologies Used

- **Python**
- **Tkinter** - GUI development
- **Requests Library** - API communication
- **OpenWeather API** - Real-time weather data
- **JSON** - Data processing
